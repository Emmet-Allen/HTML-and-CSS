The 24-Hour Notebook

Using patent technology designed by scientist in NASA, the 24-Hour Notebook is used to share any document and distribute information with the gurantee that the information will only be stored for 24-Hours after its first use.



Design

The 24-Hour Notebook is designed with a sleek stainless steel coat, and an internal flash drive that is rigged so that all of its content will be first encrypted then deleted within after 24 hours of use.



Instructions

Following the inital prompt instructions, set a unique username and password this will then set a 24-Hour timer within the notebooks internal-drive. After the intial set up,you are able to store any type of file medium within the 24-Hour Notebook upto the limitations of the devices flashdrive space. Once the 24-Hour timer goes off in the notebook, the drive will encrypt all information and wipe all content from the drive.



Sizes

The 24-Hour Notebook comes in sizes spanning from 2GBs upto 64GBs. For more space users can order the 24-Hour Notebook XL, a 1TB external hard-drive that uses an advanced version of the 24-Hour Notebooks memory wiping capabiliaties
